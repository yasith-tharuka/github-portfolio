# Yasith Tharuka Portfolio

Modern, customized personal portfolio built with React, TypeScript, and Vite.

## Stack

- React 19
- TypeScript
- Vite 8
- Lucide React icons

## Features

- Professional hero section with strong visual hierarchy
- Animated ambient background effects
- Project cards with technology tags
- Skills section with capability chips
- Contact section with clear calls to action
- Fully responsive layout for desktop and mobile

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Customize quickly

- Edit content in `src/App.tsx`
- Edit theme and layout styles in `src/App.css`
- Edit global colors and background in `src/index.css`
